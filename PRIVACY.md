Privacy Policy for Saved E Ink Frame

Saved E Ink Frame is a personal project used to display saved Pinterest board content on a private e-ink display.

This app only requests read-only access to Pinterest boards and Pins that the user authorizes. The app does not create Pins, delete Pins, edit boards, publish content, schedule content, manage ads, or share user data with third parties.

The app may store basic Pin information locally, including Pin IDs, board names, image URLs, and seen or unseen display status. This data is stored locally for the purpose of preventing repeated display images and generating the next frame image.

No personal data is sold, shared, or used for advertising. Access tokens are stored locally by the user and should not be shared publicly.

To stop using the app, the user can delete the local project files or revoke app access from their Pinterest account settings.
